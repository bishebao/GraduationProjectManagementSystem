<h2>一、项目简介</h2>
基于ssm的毕业设计（论文）管理系统：，后端 ，集成选题模块、信息管理、流程管理、文件管理等功能于一体的系统。
<h2>二、项目技术</h2>
前端 html、jquery、echarts、AmazeUI

后端 maven、springmvc、spring、mybatis
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">

<strong>毕业设计管理系统可以分为五个模块：登录模块，选题模块，信息管理模块，流程管理模块，文件管理模块。</strong>
该系统在技术上使用Spring+SpringMVC+MyBatis整合框架作为后台开发框架，AmazeUI作为前端开发框架。 并使用Ehcache作为项目的缓存，druid作为项目的数据库连接池，使用FreeMarker实现word的导出，使用Shiro完成项目的登录认证，数据库使用的则是MySQL。
该系统可以分为三类角色：学生，教师，教务管理员。
该系统前端界面使用纯HTML实现，与后台的交互则都是通过Ajax完成。

<strong>登录模块:</strong>
登录模块主要完成教务管理员，教师，学生用户的登录功能，每个角色登录之后会看到不同的操作界面。

<strong>选题模块:</strong>
选题模块主要完成师生双选这一业务流程。

<strong>信息管理模块:</strong>
信息管理模块主要完成对各种信息的crud操作。

<strong>流程管理模块:</strong>
流程管理主要完成对开题报告，中期检查，免答辩申请，验收/答辩，论文审核，盲审的流程的管理。

<strong>文件管理模块:</strong>
文件管理模块主要对文件的上传与下载。
